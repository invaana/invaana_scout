# Invaana Scout




## Usage

```python

from invaana_scout.scout import ScoutThis

scout = ScoutThis(kw="Scientific Innovations")
scout.run()

```


## Requirements

- Chrome and Chromedriver
- MongoDB


## Installation

```bash
brew install chromedriver
brew install mongodb
pip install -r requirements.txt
```
